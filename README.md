## 开发

先决条件：
- nodejs(Version >= 16)

首先运行：
```git
git clone https://github.com/DGP-Studio/Snap.Genshin.Docs.git
```

然后在项目文件夹下运行：

```bash
npm install
npm run docs:dev
```
