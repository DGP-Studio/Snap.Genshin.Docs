# Snap Genshin 插件

::: warning 安全提示
此页面中的插件经过了我们的审核，是安全的、无破坏性的  
在使用非本列表中的拓展插件时，请注意对其安全性进行判断
:::

Snap Genshin 插件为`.dll` 格式的文件。将你下载插件解压，将获得的`.dll`文件放置于 Snap Genshin 目录下的`Plugins`文件夹内之后，完全重启 Snap Genshin 即可激活插件。
